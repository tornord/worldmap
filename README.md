## Install Babel

npm install --save-dev @babel/cli @babel/preset-react @babel/plugin-transform-modules-commonjs

.babelrc
{
  "presets": ["@babel/react"],
  "plugins": ["@babel/plugin-transform-modules-commonjs"]
}
